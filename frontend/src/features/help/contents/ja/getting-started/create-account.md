## アカウントを作成する

アカウント作成は以下の手順で行います。

1. トップページから **「新規登録」** をタップ
2. メールアドレスとパスワードを入力（または Google / Apple でサインイン）
3. 確認メールに記載されたリンクをクリックして本人確認
4. プロフィール（表示名・アイコン）を設定

> 8 文字以上のパスワードと、大文字・小文字・数字を含めることを推奨します。

### 確認メールが届かない場合

迷惑メールフォルダを確認してください。それでも届かない場合は [ログインできない](/help/troubleshooting/login-trouble) をご覧ください。
