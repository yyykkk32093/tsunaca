## tsunaca とは

tsunaca はコミュニティを運営する方々が、メンバー管理・イベント告知・お知らせ・チャット・収支管理までを一元的に行える SaaS です。

### 主な機能

| 機能           | 概要                           |
| -------------- | ------------------------------ |
| コミュニティ   | 作成・階層管理・メンバー招待   |
| アクティビティ | イベント作成・参加管理・支払い |
| お知らせ       | 重要情報の配信と既読管理       |
| チャット       | メンバー同士のやり取り         |
| 課金           | 有料プラン・サブスクリプション |

### 料金

無料プランから始められ、必要に応じて有料プランへアップグレードできます。詳細は [プラン管理](/paywall) ページをご覧ください。
