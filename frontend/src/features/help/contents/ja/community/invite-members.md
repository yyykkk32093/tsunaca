## メンバーを招待する

メンバーを招待するには、コミュニティ詳細画面の **「メンバー」** タブから **「招待リンクを発行」** を選択します。

### 招待リンクの有効期限

招待リンクは有効期限を設定できます。デフォルトは 7 日間です。

### メンバー権限の変更

既存メンバーの権限（オーナー / 管理者 / 一般）はメンバー一覧から変更できます。

| 権限     | できること                     |
| -------- | ------------------------------ |
| オーナー | すべての操作（譲渡・削除含む） |
| 管理者   | メンバー管理・コンテンツ編集   |
| 一般     | 閲覧・投稿                     |
