## コミュニティを作成する

コミュニティの作成手順は以下の通りです。

1. 画面下部の **「コミュニティ」** タブを開く
2. **「+ 作成」** をタップ
3. コミュニティ名・説明・カバー画像を設定
4. 公開範囲（公開 / 招待制）を選択
5. **「作成」** をタップ

作成後は管理画面から招待リンクを発行し、メンバーに共有できます。サブコミュニティを作成して階層化することも可能です。
