## 問い合わせ管理の使い方

ハンバーガーメニュー > **運営トップ** > **問い合わせ管理** から、ユーザーからの問い合わせを一覧で確認できます。

### フィルタ

| フィルタ   | 内容                              |
| ---------- | --------------------------------- |
| ステータス | 受付中 / 対応中 / 解決済み / 終了 |

### 詳細画面の操作

- スレッドで全メッセージを確認
- 返信を送信（送信時にユーザーへアプリ内通知が自動配信）
- ステータス変更（監査ログに記録）

### 通知

新規問い合わせは Slack（`SLACK_INQUIRY_WEBHOOK_URL` 設定時）にも通知されます。
