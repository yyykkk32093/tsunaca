## Sign up steps

1. Open the app and tap **Sign up**.
2. Enter your email and password (8+ characters).
3. Confirm via the verification email.
4. Set a display name and avatar (optional).

## Tips

- Use a strong unique password (we recommend a password manager).
- You can switch the display language anytime from the menu.
